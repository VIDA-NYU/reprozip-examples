Get to know some ReproZip use cases!

This site holds some examples and use cases from different domains that showcase ReproZip, an open source tool that enables full computational reproducibility and preservation! Youâ€™ll find more information about each particular example in its associated README when you click the "REPRODUCE THIS" button below. Instructions for reproducing the examples use the vagrant and the docker unpackers. However, any of the available unpackers can be used.
